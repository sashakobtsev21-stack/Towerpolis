Drop your background music here, named exactly:

  theme     — calm looping track (extension .ogg or .wav).

Example: Assets/Audio/Resources/Music/theme.ogg  ->  loaded as "Music/theme".
It loops automatically (AudioSource loop = on), so a seamless loop sounds best.
Case matters in a build — keep it lowercase "theme".

See docs/AUDIO_GUIDE.md for sources and import tips.
