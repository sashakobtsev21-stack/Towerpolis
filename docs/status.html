Что за проект

Современный взгляд на петлю Tower Bloxx: кран качает блок-этаж, тапаешь — он падает и должен чисто лечь на предыдущий. Идеальные постановки держат башню ровной и парашютируют новых жильцов; небрежные — дают крен и физическое шатание. Чем выше башня, тем напряжённее. Готовые башни населяют постоянный 3D-город, который ты растишь.

🎯 Уникальность (чего нет у конкурентов): daily-seed стэкер (один паттерн крана для всех в мире каждый день — «Wordle для стэкеров») + постоянный «Город» с престижем (бесконечная петля).

Детерминированная логика (счёт, грейд, сид, экономика) живёт в Unity-независимой библиотеке Towerpolis.Core с NUnit-тестами — никогда не из PhysX (честность daily-seed). Физика активна только пока блок падает; дальше — заморозка + сварка + скриптовое шатание.

Что сделано (фазы 0–5)

✓Фаза 0–1 — концепт, GDD, пиллары; Unity 6.3 URP проект, Towerpolis.Core + двойной тест-харнес, CI, ADR-ы.

✓Фаза 2 — ядро игры — кран → тап → детерминированный грейд → сварка → скриптовое шатание + juice. Гейт «весело?» пройден.

~Фаза 3 — Город — районы, daily-seed, локальные лидерборды, монеты, серии, экран «район застроен», хуки музыки районов, сетка best-N (без soft-lock). Остался on-device гейт.

~Фаза 4 — прогрессия / бесконечность — апгрейды (Магнит, Бонус города), недельные миссии, достижения, логин-календарь, заморозка серии, престиж / бесконечная петля. Core готов и оттестирован; UI частично.

~Фаза 5 — UI/UX и локализация — кастомный RU/EN Loc-сервис, HUD + MetaHud переведены, safe-area, настройки. Полный проход по экранам — впереди.

Ключевые системы

  • Ядро рана: грейд/счёт/крен/страйки, сид крана + последовательность блоков
  • Серия (combo): заполняется идеальными, любой не-идеальный сбрасывает; на 5 → +монеты и сброс
  • Город: best-N сетка, депозит → монеты → серия → награда за район
  • Престиж: звёзды = постоянный множитель жильцов, сброс цикла
  • HUD: этажи+жильцы слева (иконки), монеты справа, вертикальная combo-полоса
  • Мета-HUD: ☰ меню → Город / Бонусы / Цели / Настройки; экраны «район готов» + престиж
  • Сейв: схема v3 + forward-миграции
  • Прочее: аудио (SFX + музыка районов), VFX, темы районов, Loc (RU/EN, кириллица)
